# Dubai RTA Simulator

A web-based third-person driving practice simulator inspired by Dubai RTA road-test learning scenarios.

- Longer route so the game does not end too quickly
- Visible car with third-person camera
- Fault log showing exactly what went wrong
- Final pass/fail report with fault explanations
- More location-specific environments:
  - Al Quoz: industrial road, flyover, roundabout, service road
  - Al Qusais: residential road, school zone, signals, U-turn style practice
  - Port Jumeirah: coastal road, palms, merge, bridge, roundabout-style exit
- Mobile joystick + accelerator/brake
- Mirror and indicator buttons
- RTA-style learning prompts

## Run locally

Open `index.html` in Chrome.


## Important note

This is an independent learning simulator. It is not affiliated with, endorsed by, or an official product of Dubai RTA. The location routes are inspired practice environments, not exact official exam routes.
